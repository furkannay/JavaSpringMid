How the API works

To access data by REST API entry point is /teachers
You can send POST requests to /teachers/add to add teachers.
To get a specific teacher with id send a GET request or visit /teachers/{Insert_ID_Number}
To get all teachers you can visit or send a GET request to /teachers/all.
To delete a teacher with specific id send a DELETE request to /teachers/{Insert_ID_Number}
To replace a specific id send a PUT request to /teachers/{Insert_ID_Number}
